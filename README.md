## How to run locally

1. Open `index.html` in any modern browser (desktop or mobile).
2. Controls: tap / click / Space / Up arrow to flap.

## Deploy to GitHub Pages

1. Create a new repository on GitHub.
2. Push these files to the repo.
3. In repository Settings -> Pages, select the branch (usually `main`) and root to publish.
4. GitHub Pages will host the `index.html` as a static site.

Alternatively, use Vercel/Netlify for one-click deploys.
